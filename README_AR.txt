بوابة استبيانات قطاع العلمين
============================

الرفع على Vercel:
1. فك ضغط الملف.
2. ارفع المجلد على GitHub أو استخدم Vercel CLI.
3. اربط المشروع بـVercel بدون Build Command؛ الموقع Static وجاهز مباشرة.

الربط مع ArcGIS Hub:
1. بعد نشر الموقع على Vercel انسخ رابط الموقع.
2. أضف الرابط داخل ArcGIS Hub كزر أو عنصر Embed / Iframe.
3. استخدم رابط Vercel النهائي داخل عنصر الهاب.

الملفات المهمة:
- index.html: الصفحة الرئيسية.
- styles.css: التصميم الكامل.
- script.js: البحث والتصفية والنزول التلقائي.
- assets/geo-informatics-logo.png: اللوجو الأصلي.
